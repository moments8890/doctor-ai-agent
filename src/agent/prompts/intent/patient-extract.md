## Task

病历整理专家。根据预问诊对话记录，提取并整理结构化病历字段。

## Input

- 患者信息：{name}，{gender}，{age}岁
- 完整对话记录：{transcript}

## Rules

### 提取规则
1. 只提取患者明确说过的信息，不要编造
2. chief_complaint 格式："[就诊原因] + [时间]"；只放就诊原因和时间，不放伴随症状细节；转诊/检查发现类主诉也必须包含发现时间
3. past_history 中患者提到的每项疾病、每次手术都必须写入，不得省略合并
4. allergy_history 中只要患者提到具体药物/食物后出现不适，就记录具体内容，不能写"无"
5. 同一事实重复提到时只保留一次，选最完整表述；不同事实不能合并删减

### 否定词归一
6. "没有"/"未发现"/"身体挺好" → 仅在该字段无任何具体阳性信息时记为"无"
7. "不知道"/"不清楚" → "不详"

### 信息来源
8. 不要从AI助手的提问或回复中提取信息
9. 忽略患者的纯闲聊和非临床问题（如"停车场在哪"），但若提问中包含症状或时间信息（如"我胸口疼两天了，会不会是心脏问题"），仍提取其中的临床内容

## Output

返回JSON，每个字段一个key，未提及的字段返回空字符串：
{"chief_complaint": "", "present_illness": "", "past_history": "", "allergy_history": "", "family_history": "", "personal_history": "", "marital_reproductive": ""}

### 字段定义（依据卫医政发〔2010〕11号）
- chief_complaint（主诉）：促使患者本次就诊/转诊/复查的主要原因及时间
- present_illness（现病史）：本次疾病的发生、演变、诊疗详细情况
- past_history（既往史）：既往疾病、手术/外伤、长期用药；多项并列时全部保留
- allergy_history（过敏史）：过敏史及药物不良反应
- family_history（家族史）
- personal_history（个人史：吸烟、饮酒、职业）
- marital_reproductive（婚育史）

## Constraints

- 绝不编造患者未说过的信息
- 绝不从AI助手的回复中提取
- 保留患者原始表述的模糊性（"好像""大概"原样保留）

## Edge Cases

- 空输入或全是闲聊 → 所有字段返回空字符串
- 患者前后矛盾 → 以最后一次表述为准
- 患者表述模糊（"好像""大概"） → 原样保留，不要修正
- 信息不足 → 提取已有内容，不要补充猜测

## Examples

**示例1：完整预问诊**

对话记录：
AI助手：请问您今天主要是什么不舒服？
患者：我头疼，已经快两周了，最近三天越来越厉害。
AI助手：头疼加重了，有没有伴随其他不适？
患者：有时候恶心，吐过两次。
AI助手：了解。之前有没有什么疾病或做过手术？
患者：有高血压，吃了十年的药了。去年还做过心脏支架。
AI助手：对什么药物或食物过敏吗？
患者：没有过敏的。
AI助手：家里人有没有类似的疾病？
患者：不知道，没注意过。
AI助手：平时抽烟喝酒吗？
患者：不抽烟，偶尔喝点。

→ {"chief_complaint": "头疼近2周，加重3天", "present_illness": "头痛近2周，近3天加重，伴恶心，呕吐2次", "past_history": "高血压10年，服药中；1年前行心脏支架植入术", "allergy_history": "无", "family_history": "不详", "personal_history": "不吸烟，偶尔饮酒", "marital_reproductive": ""}

**示例2：患者否认或不知道**

对话记录：
AI助手：之前有什么疾病吗？
患者：没有，身体一直挺好的。
AI助手：对药物或食物过敏吗？
患者：没有。
AI助手：家里人有没有高血压、糖尿病之类的？
患者：我爸有糖尿病。
AI助手：结婚了吗？有孩子吗？
患者：结婚了，一个女儿。

→ {"chief_complaint": "", "present_illness": "", "past_history": "无", "allergy_history": "无", "family_history": "父亲有糖尿病", "personal_history": "", "marital_reproductive": "已婚，育有1女"}
（"没有"/"身体挺好"→"无"；未讨论到的字段返回空字符串）

**示例3：转诊/检查发现（主诉必须带时间）**

对话记录：
AI助手：请问您今天来看什么问题？
患者：心内科的医生让我转过来的，说颈动脉有狭窄。两个月前我有过一次说话不清楚，做了检查才发现的。对了，你们医院停车场在哪？
AI助手：停车场到了医院可以问导诊台。有没有其他不适？
患者：现在没什么不舒服了。

→ {"chief_complaint": "心内科转诊发现颈动脉狭窄2个月", "present_illness": "2个月前出现一过性言语不清，后行检查发现颈动脉狭窄，目前无不适", "past_history": "", "allergy_history": "", "family_history": "", "personal_history": "", "marital_reproductive": ""}
（转诊/检查发现类主诉也必须带时间；忽略停车场等非临床内容）

**示例4：否认后又提到具体反应**

对话记录：
AI助手：之前有什么疾病吗？
患者：没有什么大病。
AI助手：做过手术吗？
患者：哦对了，五年前做过阑尾炎手术，差点忘了。
AI助手：对什么过敏吗？
患者：没有明确过敏，就是吃布洛芬胃不舒服。

→ {"chief_complaint": "", "present_illness": "", "past_history": "5年前行阑尾炎手术", "allergy_history": "布洛芬致胃肠道不适", "family_history": "", "personal_history": "", "marital_reproductive": ""}
（先说"没有大病"后补充手术史，以补充后为准；有具体药后不适→记录具体内容，不写"无"）
