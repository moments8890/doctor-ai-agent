# 神经外科鉴别诊断指导

本技能文件定义AI辅助诊断的输出格式和神经外科关键临床模式。
AI建议仅供参考，最终诊断由医生决定。

## 输出格式

请严格按照以下结构输出诊断建议：

### 鉴别诊断（differentials）
对每个可能的诊断，提供：
- **诊断名称**：标准医学术语
- **可能性**：低 / 中 / 高
- **推理依据**：基于患者症状和体征的一句话说明

按可能性从高到低排列，最多5个鉴别诊断。

### 检查建议（workup）
对每项建议的检查，提供：
- **检查名称**：具体检查项目（如"头颅MRI增强扫描"）
- **理由**：为什么需要此检查
- **紧急程度**：常规 / 紧急 / 急诊

### 治疗方向（treatment）
对每项治疗建议，提供：
- **药物类别**：仅提供药物类别（如"糖皮质激素"），不提供具体剂量
- **干预方式**：手术 / 药物 / 观察 / 转诊
- **说明**：简要治疗方案描述

注意：不提供具体药物剂量，因系统尚未收集过敏史和用药史。

### 危险信号（red flags）
列出需要立即处理的紧急发现。

### 免责声明
每次输出必须包含：「AI建议仅供参考，最终诊断由医生决定。」

## 不可遗漏的关键模式

以下模式在神经外科临床中必须优先考虑，漏诊可能导致严重后果：

1. **突发剧烈头痛（"雷击样"）**
   → 首先排除蛛网膜下腔出血（SAH），需急诊CT + 腰穿

2. **进行性头痛 + 视乳头水肿**
   → 颅内占位性病变（脑膜瘤、胶质瘤等），需增强MRI

3. **GCS下降 + 瞳孔不等大**
   → 脑疝，需紧急手术评估

4. **马尾综合征（鞍区麻木 + 尿潴留 + 双下肢无力）**
   → 急诊手术指征，延误可致永久性损害

5. **4.5小时内急性肢体无力 + 言语障碍**
   → 急性脑卒中，处于溶栓/取栓时间窗，需急诊CT排除出血

6. **脊髓压迫（双侧肢体无力 + 感觉平面 + 括约肌功能障碍）**
   → 紧急MRI脊柱全段，评估手术减压指征

7. **局灶性起始的癫痫 + 影像学结构性病变**
   → 考虑手术评估（肿瘤、海绵状血管瘤、AVM等）

8. **三叉神经痛非典型特征（双侧、年轻患者、感觉减退）**
   → 排除后颅窝占位（听神经瘤、脑膜瘤），需MRI

9. **脑积水三联征（步态障碍 + 认知下降 + 尿失禁）**
   → 正常压力脑积水（NPH），可通过腰穿试验评估分流手术

10. **外伤后清醒间期（伤后一度好转，后迅速恶化）**
    → 硬膜外血肿，需急诊CT + 手术

11. **NIHSS ≥ 4分 + 发病在4.5小时内**
    → 急性缺血性脑卒中，评估静脉溶栓适应证

12. **持续性癫痫发作 > 5分钟（癫痫持续状态）**
    → 急诊处理，苯二氮卓类药物优先，同时查找病因

13. **颈部外伤后四肢无力或感觉异常**
    → 颈椎损伤，制动 + 紧急影像学检查
