PCI术后抗血小板治疗方案

双联抗血小板治疗（DAPT）

PCI术后常规给予阿司匹林100mg/日 + 氯吡格雷75mg/日（或替格瑞洛90mg bid），DAPT疗程至少12个月。

高出血风险患者

高出血风险患者（HBR）可考虑缩短DAPT至1-3个月后改为单药抗血小板治疗。HBR定义包括：年龄≥75岁、eGFR<30、血红蛋白<11g/dL、近期出血史。

质子泵抑制剂（PPI）联用

DAPT期间常规联用PPI（如泮托拉唑40mg/日）预防消化道出血，尤其是有消化道溃疡病史的患者。注意：奥美拉唑可能影响氯吡格雷的抗血小板活性，优先选用泮托拉唑或雷贝拉唑。

监测要点

  • 术后48小时内观察穿刺部位有无血肿、渗血
  • 每月复查血常规，关注血小板计数和血红蛋白变化
  • 警惕黑便、牙龈出血等隐匿性出血表现
  • 术后1个月、3个月、12个月复查冠脉造影或CTA评估支架情况
