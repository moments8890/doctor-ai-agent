# 软件著作权（软著）申请指南

软著是华为应用市场上架的必要材料。全称：「计算机软件著作权登记证书」。

## 申请渠道

### 方式一：自行申请（免费，1-2 个月）

登记机构：[中国版权保护中心](https://www.ccopyright.com.cn/)

1. 注册并登录「中国版权保护中心」网站
2. 进入「软件著作权登记」→「在线申报」
3. 填写申请表并上传材料
4. 等待审核（约 30-60 个工作日）

### 方式二：代办服务（500-1500 元，3-7 天）

淘宝/1688 搜索「软著代办」，选择信誉好的服务商。代办服务商会帮你：
- 撰写软件说明书
- 整理源代码文档
- 提交并跟进审核

> **建议**：如果时间紧迫，推荐使用代办服务。

## 需要准备的材料

### 1. 软件基本信息

| 字段 | 填写内容 |
|------|---------|
| 软件全称 | 鲸鱼随行智能医疗助手软件 |
| 软件简称 | 鲸鱼随行 |
| 版本号 | V1.0 |
| 开发完成日期 | （实际完成日期） |
| 首次发表日期 | （计划发布日期，可与完成日期相同） |
| 开发方式 | 独立开发 |
| 权利取得方式 | 原始取得 |
| 权利范围 | 全部权利 |
| 软件用途 | 医疗辅助/健康管理 |
| 运行环境 | Android 6.0 及以上 |
| 编程语言 | JavaScript, Python |
| 源程序量 | （统计实际代码行数） |

### 2. 源代码文档

要求：
- 提供前 30 页和后 30 页源代码（每页不少于 50 行）
- 如果代码不足 60 页，全部提供
- 去掉空行和注释
- A4 纸格式，每页有页码

生成源代码文档的快捷方式：

```bash
# 统计代码行数
find frontend/web/src -name "*.jsx" -o -name "*.js" | xargs wc -l
find src -name "*.py" | xargs wc -l

# 导出前端代码（去注释、去空行）
find frontend/web/src -name "*.jsx" -o -name "*.js" | \
  xargs cat | grep -v '^\s*$' | grep -v '^\s*//' | grep -v '^\s*\*' \
  > software_code_frontend.txt

# 导出后端代码
find src -name "*.py" | \
  xargs cat | grep -v '^\s*$' | grep -v '^\s*#' \
  > software_code_backend.txt
```

### 3. 软件说明书

15-30 页的文档，包含：
- 软件概述（功能、用途、技术特点）
- 安装说明
- 主要功能界面截图及说明
- 操作流程说明

建议内容大纲：

```
1. 引言
   1.1 软件概述
   1.2 运行环境
2. 功能说明
   2.1 医生登录
   2.2 患者管理（新建、查看、搜索）
   2.3 病历记录（创建、编辑、查看）
   2.4 AI 辅助诊断
   2.5 任务管理
   2.6 设置
3. 操作界面说明（附截图）
4. 技术特点
```

### 4. 身份证明

- 个人：身份证复印件
- 企业：营业执照副本复印件 + 法人身份证复印件

## 费用

| 渠道 | 费用 | 时间 |
|------|------|------|
| 自行申请 | 免费 | 30-60 个工作日 |
| 代办（普通） | 500-800 元 | 20-30 个工作日 |
| 代办（加急） | 1000-1500 元 | 3-7 个工作日 |

## 注意事项

- 软著申请主体必须与华为开发者认证主体一致
- 软件名称建议与应用商店名称一致或包含关键词
- 软著证书没有有效期限制，一次申请永久有效
- 建议尽早申请，这通常是上架流程中耗时最长的环节
