# 商店素材准备

## 必需素材

### 1. 应用图标

| 规格 | 用途 |
|------|------|
| 216 x 216 px, PNG | 华为商店列表图标 |

要求：
- 圆角由商店自动处理，提供方形图即可
- 背景不能是纯透明
- 不能包含「NEW」「HOT」等营销文字

> 应用内图标已在 `android/app/src/main/res/mipmap-*` 目录下，但商店图标需要单独制作高清版。

### 2. 应用截图

| 规格 | 数量 |
|------|------|
| 1080 x 1920 px（竖屏）或 1920 x 1080 px（横屏） | 最少 3 张，最多 8 张 |

建议截图内容（按顺序）：
1. 登录页面
2. 患者列表（展示核心功能）
3. 患者详情 / 病历记录
4. AI 辅助功能界面
5. 任务管理

截图技巧：
- 使用 Android Studio 模拟器或真机截图
- 确保截图中有真实数据（用测试数据，不要空页面）
- 可以在截图外添加文字说明（制作宣传图）

### 3. 应用描述

#### 简介（一句话，≤80 字）

```
鲸鱼随行 — AI 驱动的智能医疗工作台，帮助医生高效管理患者、病历和诊疗任务。
```

#### 详细描述（≤4000 字）

```
【产品介绍】
鲸鱼随行是一款面向医疗专业人员的智能工作台应用。通过 AI 技术辅助医生完成
日常诊疗工作，提升工作效率和医疗服务质量。

【核心功能】
• 患者管理：快速创建和查找患者档案，一站式管理患者信息
• 病历记录：结构化病历录入，支持语音和文字输入，AI 自动整理
• 智能辅助：AI 辅助分析患者信息，提供参考建议
• 任务管理：诊疗任务跟踪，不遗漏任何待办事项
• 多端协作：支持手机端和电脑端同步使用

【产品特色】
• 为中国医生量身定制，符合国内医疗工作流程
• AI 辅助分析，减少重复性工作
• 数据安全加密，保护医疗信息
• 简洁易用的界面设计

【适用场景】
• 门诊接诊记录
• 患者随访管理
• 病历资料整理
• 多患者并行管理

【声明】
本应用中 AI 生成的内容仅供参考，不构成医疗诊断依据。
最终诊疗决策应由具有执业资格的医生做出。
```

### 4. 分类和标签

| 字段 | 建议值 |
|------|--------|
| 一级分类 | 医疗健康 |
| 二级分类 | 健康管理 / 医疗服务 |
| 标签 | 医疗, 健康管理, AI, 病历, 医生工具 |
| 内容分级 | 所有人 |

> 注意：如果选择「医疗器械」或「在线诊疗」子分类，可能需要提供额外资质证明。
> 建议选择「健康管理」或「医疗服务」以减少资质要求。
