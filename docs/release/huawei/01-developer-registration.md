# 华为开发者账号注册

## 1. 注册华为账号

访问 https://developer.huawei.com/consumer/cn/

点击右上角「注册」，使用手机号或邮箱注册华为账号。

## 2. 实名认证

登录后进入「管理中心」→「认证信息」，选择认证类型：

### 个人开发者

需要材料：
- 身份证正反面照片
- 手持身份证照片
- 银行卡信息（用于验证身份）

审核时间：1-2 个工作日

### 企业开发者（推荐）

需要材料：
- 企业营业执照扫描件
- 法人身份证正反面
- 企业对公银行账户信息
- 授权书（如非法人本人操作）

审核时间：1-3 个工作日

> **建议**：如果有企业资质，优先使用企业开发者。企业开发者审核通过率更高，
> 且健康管理类应用通常需要企业资质。

## 3. 创建应用

认证通过后：

1. 进入 [AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)
2. 点击「我的应用」→「新建应用」
3. 填写基本信息：

| 字段 | 填写内容 |
|------|---------|
| 应用名称 | 鲸鱼随行 |
| 应用包名 | com.doctorai.app |
| 应用分类 | 应用 → 健康管理 |
| 设备类型 | 手机 |
| 默认语言 | 简体中文 |

4. 点击「确定」创建应用

## 4. 获取应用信息

创建后记录以下信息（后续提交需要用到）：

- **应用 ID**（AppGallery Connect 分配）
- **应用包名**：`com.doctorai.app`

## 注意事项

- 华为开发者注册免费，无需缴费
- 同一企业只需注册一次，可发布多个应用
- 实名认证信息必须与软著申请主体一致
- 健康管理类应用可能需要额外资质审核（见提交指南）
