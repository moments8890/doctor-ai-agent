核心流程

新建患者

两种方式创建患者档案，满足不同场景需求。

  • 对话创建 — 自然语言输入，AI自动提取信息
  • 导入创建 — 上传病历PDF或转发聊天记录，AI解析建档
  • 信息确认 — AI提取后可直接创建或编辑完善
  • 快捷操作 — 创建后无缝衔接预约、处方等

患者列表

简洁的患者管理视图，搜索栏 + 新建入口 + 按最近交互排序的患者列表。

  • 搜索栏：按姓名、病症、编号实时筛选
  • "新建患者"作为列表第一项（虚线绿色边框）
  • 点击"新建患者" → 跳转AI对话，输入患者信息
  • 姓氏头像（微信联系人风格）
  • 红点 = 危急，橙点 = 需关注
  • 点击患者行 → 进入患者详情

患者详情

点击患者进入详情页，查看完整病历信息，推入式导航隐藏底部标签栏。

  • 基本信息卡：姓名、性别、年龄、手机号、状态标签
  • 诊断标签：灰色背景，水平排列
  • 菜单列表：就诊记录、处方、检验报告、过敏信息
  • 最近就诊摘要
  • 操作按钮：新建任务 + 导出PDF（同行）
  • AI对话咨询（独占大按钮，主操作）

Screen 4

任务管理

预约和待办事项的统一管理，支持分段切换、快速操作和详情编辑。

  • 分段控制：今天 / 待办 / 已完成
  • 预约：按时间排列，当前预约标记"进行中"
  • 待办事项：勾选框（红色=紧急，灰色=普通）
  • 左滑 → 快速"完成"或"取消"
  • 点击任务 → 详情页（编辑/完成/取消）
  • 已完成：删除线 + 时间戳
