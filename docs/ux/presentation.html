工作流

新建患者（对话驱动）

保持Chat-First一致性，通过对话创建患者，AI自动提取信息。

  • 步骤1：医生在AI对话中输入患者信息（文字或语音）
  • 步骤2：AI提取信息，生成预览卡片
  • "确认创建" → 直接保存（快速路径）
  • "编辑完善" → 打开预填表单，补充更多信息
  • 患者列表"新建患者" → 也跳转AI对话

实际对话演示

新建患者 — 完整对话

医生与AI助手的真实交互过程，展示对话驱动的工作流。

  • 1. 自然语言输入 — 医生用日常口语描述患者信息
  • 2. AI智能提取 — 从对话中提取结构化数据
  • 3. 确认或补充 — 可直接创建或编辑完善
  • 4. 创建成功 — AI提供快捷操作按钮
  • 5. 连续操作 — 同一对话中继续安排复诊

患者列表

简洁的患者管理视图，搜索栏 + 新建入口 + 按最近交互排序的患者列表。

  • 搜索栏：按姓名、病症、编号实时筛选

  • 红点 = 危急，橙点 = 需关注
  • 点击患者行 → 进入患者详情

任务管理

预约和待办事项的统一管理，支持分段切换、快速操作和详情编辑。

  • 分段控制：今天 / 待办 / 已完成
  • 预约：按时间排列，当前预约标记"进行中"
  • 待办事项：勾选框（红色=紧急，灰色=普通）
  • 左滑 → 快速"完成"或"取消"
  • 点击任务 → 详情页（编辑/完成/取消）
  • 已完成：删除线 + 时间戳
