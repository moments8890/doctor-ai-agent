Instructor vs Raw LLM — When ## 输出格式 Is Needed

This project uses two LLM call paths. structured_call() uses instructor + Pydantic which enforces JSON schema at the code level — the prompt does NOT need ## 输出格式 or format-related constraints. llm_call() returns raw text with optional json_mode — the prompt MUST define the output schema since there's no code enforcement.
